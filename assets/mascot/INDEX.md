## 维护

新增表情时：
1. 放到对应子文件夹（emojis / busts / lifestyle / working / outfits / variants）
2. 文件名格式：`{category}-{NN}-{descriptor}.png`（descriptor 用小写连字符）
3. 更新本 INDEX.md 表格
4. 如果是新类别，先在 DESIGN.md §3 讨论

> ⚠️ 牛小匠**必须保留**：墨镜 + 牛角 + 黑色卷发。
> 可变：服装颜色（红 / 蓝 / 特殊节日色）/ 道具 / 表情。
